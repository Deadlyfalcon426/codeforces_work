# Competitive Programming Journal

This repository contains my Codeforces contest solutions, practice problems, and post-contest reflections.

I use this repo to:
- track algorithmic progress
- analyze mistakes and debugging patterns
- document problem-solving strategies
- improve competitive programming consistency over time

Most folders contain:
- contest solutions (.cpp)
- reflection notes (notes.md)
- practice/upsolve work

If you're looking for the contest reflections/notes, check one of the notes.md files in each contest folder.
